Use the box to draw! The size option modifies the number of pixels. The 'Create Anaglyph' button converts the drawing to an anaglyph, meaning the drawing can only be seen through a red lens. The padding option controls the amount of space around the dark pixels within the anaglyph. The difficulty will add more dark pixels within the padding space, making the image harder to see. The darkness option controls the color range of the reds and cyans in the anaglyph, meaning with a higher darkness the anaglyph will have a greater contrast. The image selection allows the user to input an image of a QR Code, which can then be made into an anaglyph. To be able to scan the QR Code through a red lens, I would recommend a darkness of around 50. This was made for use by the San Jose Museum of Art for a scavenger hunt.
